### string split 함수

python과 다르게 C++에서는 문자열을 split할 수 있는 함수가 없다.
문자열을 split하는 함수는 다음처럼 구현할 수 있다.


```cpp
vector<string> split(string i, char delim){
	vector<string> ret;
	istringstream ss(i);
	string tmp;
	while(getline(ss,tmp, delim)){
		ret.push_back(tmp);
	}
	return ret;
}

```
